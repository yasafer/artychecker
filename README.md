# ArtyCheck

Advanced Minecraft Mod Scanner for unheardvoice

ArtyCheck — это десктопное приложение для автоматической проверки Minecraft-модов на наличие запрещённых клиентов, подозрительных сигнатур и потенциально опасных модификаций.

## Возможности

* Проверка JAR-файлов Minecraft
* Проверка по SHA256
* Проверка по названию мода и Mod ID
* Анализ Fabric / Forge метаданных
* Поиск запрещённых сигнатур внутри классов
* Ручная проверка подозрительных модов
* Система доверенных модов
* Автоматическая синхронизация базы через GitHub
* Удобная административная панель
* История сканирований

## База данных

База поддерживает:

* Trusted Mods
* Trusted Hashes
* Banned Mods
* Banned Hashes
* Instant Ban Signatures
* Manual Review Signatures

## Важно

ArtyCheck находится в стадии активной разработки.

Некоторые функции могут изменяться между версиями, а результаты проверки рекомендуется использовать совместно с ручной модерацией.

## Beta Version

Данная версия является публичной бета-версией и предназначена для тестирования новых функций и алгоритмов обнаружения модификаций.
